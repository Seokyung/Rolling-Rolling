## 💁 프로젝트 소개

친구, 동료, 지인, 연인, 가족들과 온라인으로 롤링페이퍼를 돌려보자!

어린 시절 친구들과 종이를 돌려가며 썼던 추억의 롤링페이퍼를 즐겨보아요 :)

서로의 페이퍼에 메세지를 적고, 페이퍼에 남겨진 메세지들을 확인해 볼 수 있는 롤링페이퍼 서비스입니다!

> 개발 기간 : 2023.03 ~ 2023.06<br/>~~더 나은 서비스를 위해 무한 수정 중입니다...~~

## 🔗 배포 주소

https://rolling-rolling.web.app/

<hr/>

## 📌 주요 기능

### 📃 나만의 페이퍼 생성, 공유 기능

- 원하는 주제의 롤링페이퍼 생성 가능 (페이퍼 공개여부 설정 가능)
- 링크를 통해 페이퍼 공유 가능

### ✍️ 메세지 작성 기능

- 공개 / 비공개 메세지 작성 가능
- 메세지에 이미지 / 그림 첨부 가능

### 👤 프로필 설정 기능

- 이름, 프로필 사진 변경 가능
